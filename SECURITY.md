# Security Policy

## Supported Versions

Only the latest version of Anchorgate is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Anchorgate seriously. If you believe you have found a security vulnerability, please do NOT open a public issue.

Instead, please report the vulnerability by following these steps:

1. Send an email to the maintainer (see the README for contact information).
2. Include a detailed description of the vulnerability, steps to reproduce, and any potential impact.
3. We will acknowledge your report and provide a timeline for a fix.

We follow a responsible disclosure policy and ask that you give us reasonable time to address the issue before making it public.
